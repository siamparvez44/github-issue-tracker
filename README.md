1️⃣ What is the difference between var, let, and const?
    var = function scoped. Value can be reassigned.
    let = block scoped. Value can be reassigned.
    const = block scoped. Value cannot be reassigned.

2️⃣ What is the spread operator (...)?
    The spread operator allows an iterable (such as array) to exapands it's elements.

3️⃣ What is the difference between map(), filter(), and forEach()?
    map() - this method is used to run same function/transform for all the elements and returns same length array with modified elements.
    filter() - this method returns an array with the selected elements that passed the condition.
    forEach() - this method is used to run same functions for all of the elements passed to it and does not return anything.

4️⃣ What is an arrow function?
    an arrow function is an alternative to the regular function but uses "=>"

5️⃣ What are template literals?
